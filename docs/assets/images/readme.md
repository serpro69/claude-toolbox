# claude-toolbox — brand assets

A single mark: the Claude spark held inside a hex bolt-head — "a toolbox for Claude."
Terracotta `#D9683D` on near-black `#1F1B18`.

## Files

| File | Use |
|---|---|
| `icon-dark.svg/.png` | App icon / repo avatar (dark tile) |
| `icon-light.svg/.png` | Same mark on a light tile |
| `icon-transparent.svg/.png` | Mark only, no tile — for custom backgrounds |
| `lockup-light.svg/.png` | Horizontal logo for light READMEs |
| `lockup-dark.svg/.png` | Horizontal logo for dark READMEs |
| `favicon-16…128.png` | Browser favicons (mark stays legible to 16px) |

Wordmark is JetBrains Mono ExtraBold, outlined to vector paths — no font dependency.

## Theme-aware README embed

```html
<picture>
  <source media="(prefers-color-scheme: dark)" srcset="lockup-dark.svg">
  <img src="lockup-light.svg" alt="claude-toolbox" width="420">
</picture>
```
